Menu Responsivo Moderno

Este é um exemplo de menu responsivo usando técnicas modernas de CSS. Em desktop, o menu fica fixo no topo. Em mobile, a navegação principal vai para a parte inferior da tela, no estilo de aplicativos nativos.

Técnicas utilizadas:

  • CSS Custom Properties (variáveis)
  • CSS Nesting (aninhamento nativo)
  • Glassmorphism (backdrop-filter)
  • OKLCH para cores perceptuais
  • color-mix() para variações de cor
  • Logical Properties (margin-inline, padding-block)
  • Sintaxe moderna de Media Queries (width < 768px)
  • env() para safe-area em dispositivos com notch
  • clamp() para tipografia fluida
